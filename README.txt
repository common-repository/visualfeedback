Easily install the VisualFeedback code snippet on your website and start receiving valuable customer feedback.

== Description ==

At VisualFeedback We believe that client feedback is the key to growth so we make it easy to start receiving feedback from any website or app.
We offer fully customisable feedback forms, simple "micro" surveys, longer exit surveys and in-page feedback for emails.
Our software takes a screenshot of the relevant page to give the feedback visual context.

Easily add VisualFeedback to your website with the VisualFeedback plugin for Wordpress.

== Installation ==

1. Upload `/visualfeedback/` to the `/wp-content/plugins/` directory or add it from the plugin menu
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start a free trial with VisualFeedback if you don't already have an account
4. Obtain your website ID either during account setup or via Admin > Websites > Website ID
5. Copy your VisualFeedback website ID and paste it into the Website ID text box. Your VisualFeedback feedback button will now show.

== Frequently Asked Questions ==

= Where can I find my VisualFeedback Website ID? =

When you create a new VisualFeedback account select 'Wordpress' as your CMS type and you will be given your website ID.

= How do I sign up for VisualFeedback? =

You can start a free trial at [VisualFeedback](https://visualfeedback.cloud/signup "VisualFeedback Free Trial")

= How do I remove VisualFeedback from my site? =

Simply deactivate or remove the plugin.
